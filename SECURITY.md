# 🔐 Security Policy

## ⚙️ Supported Versions

| Version    | Supported | Notes              |
| ---------- | --------- | ------------------ |
| 2.5.3      | ✅         | Fully supported    |
| 2.0.0 or < | ❌         | Outdated, not safe |
| 1.0.0 or < | ❌         | Outdated, not safe |

## 🛡️ Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately by contacting:

* Email: [stefano.ceriani@blu.it](mailto:stefano.ceriani@blu.it)

Please include:

* A clear description of the issue
* Steps to reproduce
* Possible impact

We will respond as soon as possible and work on a fix.

## 🐛 Reporting Bugs

For non-security bugs, feel free to open a public issue and describe the problem.

Thank you for helping improve this project ♥️
